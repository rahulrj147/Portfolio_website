Examples of my work

MentorWay - Full-Stack EdTech Platform

This is a full-stack Ed-tech web application built with MongoDB, Express.js, React.js, Node.js, Tailwind CSS, Cloudinary, and Razorpay. For students, the platform allows them to register with email verification, purchase courses securely via Razorpay, stream video content seamlessly, and track their progress. They also receive regular email updates and confirmations. For instructors, it offers a dedicated dashboard to create and manage courses, monitor enrollments, and track earnings. Courses are hosted using Cloudinary, ensuring fast and optimized video delivery. The app features role-based authentication, a modern responsive design, and smooth user experience for both learners and instructors.

AI-CodeXpert - Your coding expert.

AI-CodeXpert is an intelligent code review and assistance platform built to help developers improve code quality and productivity. By integrating Gemini AI, it automatically detects syntax errors, suggests corrections, and provides smart recommendations for optimizing code structure and logic. Designed for scalability, the system handles 10,000+ lines of code in a single request while maintaining a rate limit of 15 requests per minute and 1,500 requests per day, ensuring smooth and reliable performance for high-traffic usage. The frontend, built with React, offers a clean and user-friendly interface, while Node.js and Express.js manage the backend logic and API interactions. All user data and request logs are stored securely in MongoDB, making the platform efficient, scalable, and developer-focused.

Tags
Gemini AI MongoDB Express.js Node.js React.js

CoreFit Collective – The Ultimate Exercise Companion

CoreFit Collective is a fully responsive fitness web app that helps users discover, explore, and stay motivated through categorized workout plans. Built using React and styled with Tailwind CSS, the app delivers a smooth user experience across all devices. It features body-part-based filtering, allowing users to easily find exercises that target specific muscle groups. Each exercise has a dedicated page with details like targeted muscles, equipment used, and visuals. Additionally, the app integrates YouTube video suggestions, enabling users to watch demonstrations directly, making workouts more interactive and beginner-friendly.
